# Rescue Command Center (Mobile-First)

ระบบศูนย์สั่งการกู้ภัยโทนสีแดง-ขาว-ดำ พร้อมใช้งานผ่านมือถือ มีระบบ:
- สมัครสมาชิก / ล็อกอิน
- Dashboard สรุปภาพรวม
- บันทึกข้อมูลเคส
- หน้าแจ้งเหตุสำหรับแอดมิน (ส่งแจ้งเตือนในระบบ + LINE กลุ่ม)
- รายงานย้อนหลัง
- ชุดทดสอบอัตโนมัติ

## 1) ติดตั้งทีละขั้น (แบบจับมือทำ)

1. ติดตั้ง Python 3.11+
2. เปิด Terminal แล้วเข้าโฟลเดอร์โปรเจ็กต์นี้
3. สร้าง virtual environment
   ```bash
   python -m venv .venv
   source .venv/bin/activate
   ```
4. ติดตั้งแพ็กเกจ
   ```bash
   pip install -r requirements.txt
   ```
5. (แนะนำ) ตั้งค่า Secret และ LINE token
   ```bash
   export SECRET_KEY="เปลี่ยนเป็นรหัสลับของหน่วยงาน"
   export LINE_NOTIFY_TOKEN="ใส่ token ถ้าต้องการส่งเข้าไลน์กลุ่ม"
   ```
6. รันระบบ
   ```bash
   uvicorn app.main:app --reload
   ```
7. เปิดมือถือหรือคอมเข้าเว็บ `http://localhost:8000`

## 2) วิธีใช้งานระบบ (เจ้าหน้าที่)

1. เข้า `/register` เพื่อสมัครสมาชิก
2. เข้า `/login` เพื่อล็อกอิน
3. เข้าหน้า Dashboard ดูสรุปเคสและแจ้งเตือน
4. ไปหน้า `บันทึกข้อมูลเคส` เพื่อเพิ่มเคสใหม่
5. ไปหน้า `รายงานย้อนหลัง` เพื่อดูข้อมูลย้อนหลังและกรองสถานะ

## 3) วิธีใช้งานระบบ (แอดมิน)

1. สมัครโดยเลือกบทบาท `admin`
2. ล็อกอินแล้วเข้า `หน้าแจ้งเหตุ (แอดมิน)`
3. กรอกหัวข้อเหตุ รายละเอียด และพิกัด lat/lng
4. กด `แจ้งเหตุ + ส่งแจ้งเตือน`
5. ระบบจะ:
   - เก็บเหตุการณ์ในฐานข้อมูล
   - ส่งแจ้งเตือนเข้าแอพให้ผู้ใช้ทุกคน
   - ส่งเข้า LINE กลุ่ม (ถ้าตั้ง `LINE_NOTIFY_TOKEN`)

## 4) ทดสอบทุกระบบ

```bash
pytest
```

## 5) เตรียมขึ้นโฮสต์จริง

1. ตั้งค่า environment variables บนโฮสต์
2. ใช้ฐานข้อมูลจริง (เปลี่ยน `DB_URL` เป็น PostgreSQL ได้)
3. รันด้วย process manager (เช่น systemd / docker / render / railway)
4. เปิด HTTPS และกำหนดโดเมนหน่วยงาน
